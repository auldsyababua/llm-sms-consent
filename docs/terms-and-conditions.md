**Program name:** LLM SMS Assistant

**Program description:** SMS-based personal assistant conversation; messages are processed via a large language model for individual use.

**Opt-in methods:**
- Online: end users provide their phone number during signup at https://auldsyababua.github.io/llm-sms-consent/ and check a box agreeing to receive text messages.
- SMS keyword: end users can text START to the program’s number to initiate the service.

**Message/data rates:** Message and data rates may apply.

**Message frequency:** Message frequency varies based on conversation activity.

**Support:** Reply **HELP** for help or open a GitHub issue at https://github.com/auldsyababua/llm-sms-consent/issues.

**Opt-out:** Text **STOP** to stop receiving messages.
